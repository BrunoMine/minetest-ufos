UFO = Case + Circuitos + Power Cell + Computer + Controls + 4:Light + 4:Oxigen + 4:Speaker

Engine = 1:EmDrive + 2:Circuitos

Case: 
 x 2 x	← 1 = Bloico de Ferro 
 2 3 2	← 2 = Block de Vidro
 1 4 1	← 3 = Cadeira Estofada
			← 4 = Engine

Emdrive:
 x x 1	 ← 1 = Lingote de Ferro 
 2 3 x	 ← 2 = Mola Espiral de Aço
 x x 1	 ← 3 = Câmara de Resonancia de Microondas

Mola de Ferro: 
 * 9 x Arame de Aço
 
Câmara de Resonancia de Microondas: 
 1 2 1	 ← 1 = Lingote de Ferro 
 1 3 1	 ← 2 = Bobina de Cobre
 1 2 1	 ← 3 = Mese Crystal
 
Circuitos:
 1 2 1	← 1 = Lingote de Cobre
 2 3 2	← 2 = Lingote de Bio-Plastico 
 1 2 1	← 3 = Micro processador

Lingote de Bio-Plastico:
 * 9 x Ambar
 
Ambar: extraido do tronco das árvores de selva (default:jungletree) quando colocado na forja.

Micro Processador:
 1 2 1	← 1 = Lingote de Ouro
 1 3 1	← 2 = Lingote de Bio-Plastico 
 1 2 1	← 3 = Cristal de Diamante

Power Cell:
 1 1 1	← 1 = Bloco de Aço
 2 2 2	← 2 = Pilhas/Batery (type D)
 3 3 3	← 3 = Bloco de Diamante

Pilhas/Batery (type D):
 1 2 1	← 1 = Papel
 1 3 1	← 2 = Bloco de Cobre
 1 4 1	← 3 = Bloco de Carvão
		 	← 4 = Bloco de Estanho
		 	
Computer:
 1 2 3	← 1 = Resistor
 4 5 6	← 2 = Cabos de Cobre
 7 8 9	← 3 = Capacitor
		 	← 4 = Interfece Cerebral
		 	← 5 = Micro Processador
